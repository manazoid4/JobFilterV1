# JobFilter Agents

Simple agent prompts for Claude, Codex, Gemini, or ChatGPT.

Use these files as copy/paste instructions when switching tools.

## Agents

- `builder-agent.md` — builds and fixes code
- `research-agent.md` — turns competitor/customer research into practical product moves
- `sales-agent.md` — improves landing page, pricing, and conversion copy
- `qa-agent.md` — checks broken flows before launch

## Rule

Keep agents simple. Each one should do one job only.

Do not move active app code just to fit this folder structure.
