# 🏗️ JobFilter.uk: SPEC-STRIKE (V2.0)
**The "Built in Birmingham" Action Engine for Modern Tradesmen.**

JobFilter is a multi-agent infrastructure designed to replace high-cost office staff (Scouts, Estimators, and Sales) with autonomous Google Cloud agents.

## 🚀 The Spec-Strike Workflow
1. **The Intake:** Autonomous polling of 300+ UK Council portals for residential leads.
2. **Blueprint Mining:** Gemini 1.5 Flash scans PDFs for "Technical Hooks" (Steel spans, brick types, heritage constraints).
3. **The Strike:** Automated expert-level outreach triggered at the moment of planning approval.

## 🛠️ Tech Stack
- **Frontend:** Next.js 14 / Tailwind CSS (Industrial "Dirty Hands" UI).
- **Intelligence:** Vertex AI Agent Builder & Gemini 1.5 Flash.
- **Backend:** Node.js API routes on Google Cloud Run.
- **Database:** Firestore (Real-time lead tracking).
