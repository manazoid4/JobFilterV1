# Al-Mosque — Competitor Analysis

**URL:** https://www.al-mosque.com
**Type:** Indirect competitor — open-source mosque management system

## What They Do
Al-Mosque is an open-source mosque management system. It handles:
- Prayer time calculation and display
- Donation tracking
- Member/community management
- Event scheduling
- Financial reporting

## Features
- Open-source (free to self-host)
- Prayer time automation
- Donation management
- Member database
- Event calendar
- Financial reports

## Live Translation: ❌ No
Al-Mosque does NOT offer live speech-to-text or translation of khutbahs.

## Pricing
Free (open-source, self-hosted)

## Tech Stack
- Self-hosted
- Requires technical knowledge to set up
- No SaaS offering

## khutba.io vs Al-Mosque

| Feature | Al-Mosque | khutba.io Edge |
|---|---|---|
| Mosque management | ✅ Core | ❌ Not our focus |
| Live translation | ❌ No | ✅ Core product |
| Screen display | ✅ Prayer times only | ✅ Live khutbah text |
| Setup complexity | High (self-hosted) | Low (10 min, browser-based) |
| Pricing | Free (but needs IT person) | £29/mo (no IT needed) |
| Target user | Mosque admin with tech skills | Any masjid committee |

## Key Takeaway
Al-Mosque is not a direct competitor — they solve a different problem (mosque management vs live translation). However, they already have relationships with masjids and could potentially add translation as a feature.

## Risk Level: 🟢 LOW
- Open-source = slow-moving
- Different problem space
- Self-hosted model doesn't scale well
- No live translation capability

## Action
- Monitor for any translation feature additions
- Consider building an integration/plugin for Al-Mosque
- Position khutba.io as complementary: "We handle the khutbah, Al-Mosque handles everything else"
