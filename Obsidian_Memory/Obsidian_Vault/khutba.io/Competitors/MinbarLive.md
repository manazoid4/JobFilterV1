## What They Do
Live captioning and translation of khutbahs. Congregation accesses via QR code on own phone.

## Features
- Live captioning (mic → text)
- 135+ languages
- QR code per session — congregation scans, picks language
- Web-based, no app install
- Personal device model (not masjid screen)

## Languages
Arabic · English · Urdu · German · Turkish · Russian · French · Spanish · Croatian · Albanian

## khutba.io vs MinbarLive

| Feature | MinbarLive | khutba.io Edge |
|---|---|---|
| Delivery | Personal phone only | **Masjid screens + phone** |
| Auto-scroll | Unknown | **Screen-optimised built-in** |
| RTL display | Unknown | **Arabic/Urdu RTL first-class** |
| Archive | Unknown | **Searchable khutbah archive** |
| Branding | Generic | **Masjid logo + custom theme** |
| Currency | EUR | **GBP — UK-first** |
| Target market | Bosnia/Europe | **Birmingham UK → national** |

## Key Takeaway
MinbarLive = phone-first. khutba.io = **screen-first** — big display whole congregation sees simultaneously. Different UX model, both can coexist. Our edge: screen presence, UK focus, masjid branding, archive.
