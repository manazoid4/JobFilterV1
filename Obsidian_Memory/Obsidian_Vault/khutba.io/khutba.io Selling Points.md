## Core Pitch
> "Every Muslim in your masjid understands the khutba — in their language, live, on screen."

## Top 5 Points
1. **Full inclusion** — Urdu, Somali, Bengali, English speakers all follow along
2. **Zero hardware cost** — any existing screen + laptop, open browser, done
3. **Under 10-min setup** — account → mic → languages → live
4. **Priced for masjids** — community budgets, cancel anytime
5. **Archive = institutional memory** — every khutba stored and searchable

## Objections
| Objection               | Answer                                    |
| ----------------------- | ----------------------------------------- |
| Imam already translates | Free the imam — let him focus on delivery |
| Too expensive           | Less than a box of flyers per month       |
| Tech unreliable         | Offline fallback, local network support   |
| Arabic displays wrong   | RTL built in from day 1                   |
| GDPR concerns           | Opt-in archive, UK data storage           |

## Target Customers
- Masjid committees (decision makers)
- Islamic centres with multi-ethnic congregations
- Friday Jumu'ah — largest weekly gathering
- Ramadan tarawih — high attendance
- Nikah / Janazah — formal speeches
