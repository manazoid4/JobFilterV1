## SELF-LEARNING LOOP

Each run MUST:

1. Review ALL stored research files  
2. Identify:
   - patterns that repeat
   - ideas not yet implemented
   - strongest competitor advantages  

3. Select ONLY:
   - highest impact insights (1–3 max)

4. Improve them:
   - simplify
   - remove friction
   - increase clarity
   - align with product

5. Feed into build decisions

- Can this convert better?
