# Sales Specialist Schedule

## Frequency
**Weekly -- every Wednesday**

## Time
**9:00 AM**

## Trigger
Time-based. Runs every Wednesday morning.

## Pre-conditions
- CEO Agent weekly review from Monday is available
- Research Agent output from previous Friday is available
- `Marketing/` folder has been reviewed for new notes

## Duration
Estimated 20-30 minutes.

## Output Location
`06_Agents/Sales_Specialist/outreach-YYYY-MM-DD.md`

## Input From CEO Agent
Check Monday's CEO weekly review for any directive to Sales_Specialist before running.

## Input From Research Agent
Check previous Friday's research intel for competitor or pain-point findings that can inform this week's sales angles.

## Output Feeds Into
- Content Repurposing Agent (outreach templates become content)
- Social Media Agent (sales angles become social posts)
- CEO Agent (reviewed next Monday)

## Success Check
At end of week: were any outreach messages sent? Did any get responses? Log results in `Marketing/` folder.

## Monthly Review
Review open/reply rates on all outreach sent. Identify best-performing angle. Double down on it the following month.
