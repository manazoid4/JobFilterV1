# Social Media Agent Schedule

## Frequency
**Daily**

## Time
**7:00 AM**

## Trigger
Time-based. Runs every morning at 7am, 7 days a week.

## Pre-conditions
- Vault notes from past 48 hours are accessible
- Previous day's post file exists (to check for angle repetition)
- `Marketing/` folder is accessible

## Duration
Estimated 10-15 minutes.

## Output Location
`06_Agents/Social_Media_Agent/posts-YYYY-MM-DD.md`

## Weekly Content Calendar Guidance

| Day | Recommended Angle |
|---|---|
| Monday | Motivation / industry observation |
| Tuesday | Product feature highlight |
| Wednesday | Tradesman pain point / competitor comparison |
| Thursday | Behind-the-scenes / founder story |
| Friday | Result, stat, or social proof |
| Saturday | Tip or "did you know" |
| Sunday | Lighter / engagement-bait question |

Note: These are guidelines, not rules. Follow if no stronger angle is available.

## Output Feeds Into
- Content Repurposing Agent (posts can inspire other formats)
- CEO Agent (reviewed weekly for reach and engagement data)

## Escalation
If no vault notes have been updated in the past 48 hours, draw from the full vault. The `Research/` and `Marketing/` folders always have usable material.

## Monthly Review
Review: which post types performed best? Which platform drove most traffic? Adjust calendar accordingly.
