# JobFilter / Sisyphus

Sisyphus is the structured agent output and learning sub-system. Agent-produced content, experiments, and tested patterns live here.

## Subfolders

| Folder | Contents |
|--------|----------|
| `AgentLogs/` | Raw agent run logs |
| `Brand/` | Brand asset outputs from agent runs |
| `Competitors/` | Competitor research agent outputs |
| `Content/` | Content batches produced by agents |
| `Experiments/` | Experimental agent output variants |
| `Failures/` | Failed or rejected agent run outputs |
| `Prompts/` | Prompt variants tested in agent runs |
| `Research/` | Research outputs from agent sessions |
| `SEO/` | SEO content and keyword outputs |
| `Social/` | Social media content outputs |
| `VisualDirection/` | Visual direction research outputs |
| `WinningPatterns/` | Patterns that performed well — distill these into main vault |
