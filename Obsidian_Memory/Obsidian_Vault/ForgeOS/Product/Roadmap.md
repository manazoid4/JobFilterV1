# ForgeOS — Roadmap

← [[ForgeOS Map]] · [[Product Overview]]

## Phase 1 — MVP (Now)

Goal: Prove cross-device AI session continuity works.

- Session Dashboard
- Persistent tmux sessions
- Mobile-first UI
- WebSocket terminal streaming
- Session metadata
- AI activity feed
- Notifications
- Resume Anywhere

Full detail: [[MVP Scope]]

## Phase 2 — Multi-User

- Firebase Auth / Supabase Auth
- Multi-user workspaces
- Supabase persistence (replace JSON store)
- Tailscale integration (remote nodes)
- AI provider switching
- Token usage tracking

## Phase 3 — Public Beta

- Voice control ("Resume JobFilter session")
- Native mobile app (iOS + Android)
- Public cloud hosting
- Team collaboration
- Usage billing

## Success Metrics

Phase 1: A solo user can resume an AI session on phone in < 2s
Phase 2: 10+ concurrent users with shared workspaces
Phase 3: Public launch, paying customers
