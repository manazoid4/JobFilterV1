# Competitor Intel

← [[Market Gaps]] · [[ForgeOS Map]]

## Key Competitors to Watch

### Termius
Best mobile SSH client. No AI. Syncs profiles not sessions.
Gap: would need to add AI layer + session persistence + workspace abstraction.

### Warp
Best AI terminal UI. No mobile. Desktop only.
Gap: no mobile, no remote persistence, no cross-device.

### Cursor
Most popular AI editor. Agent mode growing fast.
Gap: no supervision from outside, no mobile, desktop only.
**Most likely to enter this space — watch carefully.**

### Linear (inspiration, not competitor)
Interface inspiration. Keyboard-first, dark, high-density, no bloat.
Shows what "industrial + modern" looks like done right.

## Potential Future Threats

- Warp adds mobile client
- Termius adds AI + tmux awareness
- Cursor adds mobile supervision
- A new product specifically targeting this gap

## Timeline

Market researched: 11 May 2026. Re-research trigger: any of above ships new feature.
