# Multi-Model Pipelines

## Core Pattern
- Claude handles visual/product judgement.
- DeepSeek handles cheap backend-heavy work.
- Codex handles repo execution, verification, and review.

## JobFilter Use
- Use cheap models only for non-sensitive drafts and batch thinking.
- Keep lead data, customer data, keys, and paid logic out of untrusted tools.
- Best fit: test generation, notes, low-risk refactors, rough analysis.
