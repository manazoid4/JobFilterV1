# AI Cost Optimization

## Rules
- Spend expensive models on judgement, not bulk work.
- Use cheaper models for repeatable drafts, test ideas, and low-risk scripts.
- Keep private code, customer data, and API keys out of weak-trust tools.

## JobFilter Link
- Cost saving only matters if lead quality stays high.
- Never reduce model cost by weakening paid lead quality.
