## Current Direction

AgentDock is a plug-and-play agent operations platform for businesses running workflows through ServiceNow, Zendesk, Jira, or custom ticketing systems.

It drops agents into existing processes without replacing them.

First build: Complaint management workflow on ServiceNow.

## Open Decisions

- ServiceNow or Zendesk connector first.
- Self-hosted MVP or cloud-hosted.
- OpenAI first or both OpenAI + Anthropic from day one.
- Whether to build a connector SDK for third-party developers.

## Rule
- Build for visibility first.
- Every agent must show status without opening logs.
- Every action must be traceable in an audit log.
- Every critical action needs human approval.
- No hidden autonomy.
- Compliance-ready by default.
