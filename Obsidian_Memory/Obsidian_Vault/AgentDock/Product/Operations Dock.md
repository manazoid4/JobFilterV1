# Operations Dock

The main workspace view.

Shows every active agent, connected system, and current workflow status across the entire business.

## What it shows

- Active agents and their current tasks
- Connected systems (ServiceNow, Zendesk, Jira, etc.) with connection status
- Active workflows and their stage
- Approval queue — items waiting for human review
- Recent handoffs and system updates
- Alert feed — errors, escalations, compliance flags

## UX rule

Everything visible. No clicking through to find status. A manager should see the state of all agent operations in 10 seconds.
