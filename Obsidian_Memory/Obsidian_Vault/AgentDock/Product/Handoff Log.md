# Handoff Log

Timeline of agent actions with full audit trail.

## Every handoff includes

- What was done
- What changed in the connected system
- What needs human review
- What system to update next
- Compliance notes
- Timestamp and agent identity

## Audit trail

Every action is logged immutably:

- Agent identity and model
- Input received
- Decision made
- Output produced
- System updated
- Human approval (if required)

## UX rule

Every handoff must be readable by a human in under 30 seconds. Auditors must be able to trace any action back to its source.
