# Agent Board

Cards for each active agent.

## What each card shows

- Name
- Role (complaint triage, response draft, escalation, etc.)
- Model (GPT-4, Claude, etc.)
- Current workflow
- Connected system
- Status (idle, running, waiting for approval, error)
- Last action and timestamp

## UX rule

Status must be readable without opening logs. A card should tell you if an agent is working, stuck, or waiting for a human.
