# Product Outline

## Working idea

A simple web app inspired by Plaud Note, but built for text-first idea capture and LLM routing.

## Problem

People have useful ideas in the wrong moment:

- while walking
- while driving
- while doing another task
- while talking to someone

They do not want to start a big workflow.
They want to save the thought in seconds and act on it later.

## Product job

1. Capture a thought quickly.
2. Classify it lightly.
3. Route it to the right LLM provider.
4. Keep a searchable idea history.
5. Turn rough notes into drafts, prompts, plans, and summaries.

## Simple feature set

- one-click capture
- quick title + body
- voice-to-text later if needed
- model picker
- prompt templates
- saved idea inbox
- export/share link
- follow-up reminders

## LLM integrations

Design around connectors for major providers:

- OpenAI
- Anthropic
- Google
- Mistral
- xAI
- Perplexity

## User promise

Stop losing thoughts.
Stop rewriting the same prompt.
Stop deciding where to send the idea.

Save once. Push to the right model. Move on.
