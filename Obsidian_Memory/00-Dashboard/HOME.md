## PRODUCT CONTEXT

JobFilter is not a job board.

It is an intake system:

1. Find real work.
2. Filter out weak matches.
3. Score useful leads.
4. Deliver only what is worth acting on.

Core promise:

- REAL LEADS
- NO CHASING
- NO COMPETING
- STAY IN CONTROL
