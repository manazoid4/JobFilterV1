# Product Context

## What JobFilter Is

JobFilter is an intake system for UK tradesmen.

It finds real construction signals, filters noise, scores opportunities, and helps tradesmen act only on work worth chasing.

## What It Is Not

- Not a generic job board.
- Not a demo tool.
- Not a list of fake leads.
- Not a marketing dashboard.

## Core Promise

- REAL LEADS
- NO CHASING
- NO COMPETING
- STAY IN CONTROL

## Priority Order

1. Lead quality.
2. Data pipeline reliability.
3. Speed of delivery.
4. Monetisation hooks.
5. UX simplicity.
