# Obsidian Graph Rules

## Strict Linking Rules
1. Project can link to Area + Resource only.
2. Area can link to Resource only.
3. Resource should rarely link out.
4. No random cross-linking.
5. If unsure, do not link.

## PARA Meaning
- Project = active outcome with finish line.
- Area = ongoing responsibility.
- Resource = reference material.
- Archive = inactive or dead.
